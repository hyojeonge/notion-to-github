# 클라우드 서비스 제공업체

---

## 클라우드 서비스 제공 업체

- AWS : 약 32%
- MS Azure : 약 22%
- GCP : 약 11%
- 국내에서는 네이버 클라우드, KT 클라우드, NHN 클라우드 등
## AWS

- 아마존에서 제공하는 클라우드 컴퓨팅 플랫폼
- 주요 서비스 : EC2(Elastic Compute Cloud), S3(Simple Storage Service), RDS(Relational Database Service), Lambda, SageMaker 등
## MS Azure

- 마이크로소프트에서 제공하는 클라우드 컴퓨팅 플랫폼
- 주요 서비스 : Azure Storage, Azure SQL Database, Azure Functions, Azure AI 등
## GCP

- 구글에서 제공하는 클라우드 컴퓨팅 서비스
- 구글 검색, 유튜브, Gmail 등 구글의 서비스에서 사용되는 것과 동일한 인프라를 기반으로 다양한 클라우드 컴퓨팅 서비스 제공
- 주요 서비스 : Compute Engine, Cloud Storage, BigQuery, Vertex AI 등
- VM웨어, 레드햇, 델 테크놀로지스, HPE
## VM웨어

- 가상화 기술의 선두 주자로서, 프라이빗 클라우드 구축에 필수적인 가상화 솔루션 제공
- 대표 제품군 : vSphere, vSAN, NSX 등
## 레드햇

- 오픈소스 기반의 프라이빗 클라우드 솔루션 제공
- Openshift를 통해 컨테이너 기반의 프라이빗 클라우드 환경을 구축할 수 있도록 지원
## 델 테크놀로지스

- 하드웨어 인프라와 소프트웨어 솔루션을 통합하여 프라이빗 클라우드 구축에 필요한 종합적인 솔루션 제공
- 대표 제품 : VxRail, PowerEdge 등, 대표 제품군을 통해 HCI 기반의 프라이빗 클라우드 환경을 구축할 수 있도록 지원
## HPE

- 그린레이크(GreenLake)와 같은 서비스를 통해 프라이빗 클라우드 솔루션을 제공
- 하드웨어, 인프라, 소프트웨어 및 서비스를 결합하여 기업이 자체 데이터센터에서 클라우드와 유사한 경험을 얻을 수 있도록 지원
## 국내 프라이빗 클라우드 기업

- 오케스트로, 이노그리드, NHN인재, 에이블클라우드, 나무기술, 효성인포메이션세스템 등
## MSP

- Managed Service Provider
- 기업의 IT 시스템 운영 및 관리를 전문적으로 대행하는 서비스를 제공하는 기업
