## 클라우드 이용 모델

- 클라우드 컴퓨팅 서비스를 사용하는 방식
- 각 모델은 제공되는 서비스 범위와 사용자의 관리 책임 범위가 다름
- 대표적인 이용 모델은 제공하는 방식에 따라 IaaS, PaaS, SaaS 등이 있음
## as a Service

- 클라우드 컴퓨팅 환경에서 다양한 IT 자원과 서비스를 필요에 따라 제공하는 방식
- 주요 유형 : IaaS, PaaS, SaaS 등
## IaaS

- Infrastructure as a Service(서비스형 인프라)
- 서버, 스토리지, 네트워크 등 IT 인프라 자원을 인터넷을 통해 필요한 만큼 빌려 사용하는 서비스
- 사용자는 물리적인 하드웨어를 직접 구축하고 관리할 필요 없이, 클라우드 제공업체의 인프라를 활용하여 IT 환경을 구축하고 운영할 수 있음
## PaaS

- Platform as a Service(서비스형 플랫폼)
- 애플리케이션 개발, 실행, 관리에 필요한 플랫폼을 클라우드 환경에서 제공하는 서비스
- 활용 사례 : 웹 애플리케이션 개발 및 배포, 모바일 애플리케이션 개발, API 개발 및 관리, 빅데이터 분석 등 
## SaaS

- Software as a Service(서비스형 소프트웨어)
- 소프트웨어 애플리케이션을 인터넷을 통해 제공하는 서비스
- 활용 사례 : 이메일, 협업 도구, CRM, 온라인 오피스 도구 등
## FaaS

- Function as a Service(서비스형 함수)
- 서버리스 컴퓨팅의 핵심 기술
- 개발자는 서버 인프라를 관리할 필요 없이 특정 이벤트에 반응하는 코드(함수)를 작성하고 실행할 수 있음
- 활용 사례 : API 개발, 실시간 데이터 처리, 이미지.비디오 처리, 챗봇, IoT 백엔드 처리
## XaaS

- Everything as a Service(서비스형 모든 것)
- 클라우드 컴퓨팅 환경에서 제공되는 다양한 서비스 모델을 포괄하는 용어
- IT 자원 뿐만 아니라 다양한 서비스와 솔루션을 인터넷을 통해 제공하는 것
- 주요 유형 : IaaS, PaaS, SaaS, FaaS, DaaS(Desktop as a Service), DBaaS(Database as a Service), SECaaS(Security as a Service) 등
